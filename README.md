# 研词荧光

一个面向考研英语阅读复盘的高光生词整理网页。页面支持上传卷面照片、模拟整理荧光笔标出的生词，并用电子卡片进行复习。

## 在线预览

https://magicalxdddd-ai.github.io/highlight-word-recognizer/

## 主要功能

- 上传或预览试卷照片
- 模拟提取荧光笔标出的英语生词
- 按来源和标签整理待复习词
- 点击词条切换待复习 / 已掌握状态
- 使用电子卡片查看单词、释义和例句
- 手动补录漏识别的单词

## 技术实现

- HTML / CSS / JavaScript
- FileReader 本地图片预览
- 原生 DOM 状态渲染
- 响应式布局，适配电脑和手机
- GitHub Pages 静态部署

## 本地使用

直接打开 `index.html` 即可预览。仓库根目录下的 `index.html`、`styles.css`、`app.js`、`og.png` 和 `.nojekyll` 是 GitHub Pages 使用的静态网站文件。
