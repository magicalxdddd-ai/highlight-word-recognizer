# 研词荧光

考研英语阅读高光生词提取与电子复习网站。

## 网站地址

[https://magicalxdddd-ai.github.io/highlight-word-recognizer/](https://magicalxdddd-ai.github.io/highlight-word-recognizer/)

## 这个网站能做什么

- 上传或预览试卷照片
- 模拟提取荧光笔标出的英语生词
- 把生词整理成复习词库
- 用电子卡片查看来源、标签和释义
- 手动补录漏识别的单词

## 本地预览

```bash
npm install
npm run dev
```

然后打开：

[http://localhost:3000/](http://localhost:3000/)

## GitHub Pages

仓库根目录下的 `index.html`、`styles.css`、`app.js`、`og.png` 和 `.nojekyll` 是 GitHub Pages 使用的静态网站文件。
